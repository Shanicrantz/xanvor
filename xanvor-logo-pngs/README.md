──────────────────────────────────────────────────────────────────
01-NV-Ligature/    (mark only, no wordmark)
──────────────────────────────────────────────────────────────────
The signature NV ligature alone — for embossing, embroidery, wax
seals, app icons, and as a small badge anywhere XANVOR text already
appears nearby.

Sizes per variant: 256, 512, 1024, 2048 px
Variants: 12 color/background combinations

USE FOR:
- App icon / launcher icon
- Embroidery files (give to embroiderer at 2048px)
- Wax seal die (give to seal maker)
- Brass foil stamping die (give to printer)
- Watermark on photography

──────────────────────────────────────────────────────────────────
02-Stacked-Lockup/    (mark on top, XANVOR below)
──────────────────────────────────────────────────────────────────
The primary brand lockup — mark above wordmark. Use this whenever
there's vertical space (square or portrait crops).

Sizes: 800, 1600, 3200 px wide
Variants: 8 color combinations

USE FOR:
- Instagram posts (square)
- Packaging boxes (centered top)
- Business card front (already printed, but reference this)
- Hero sections of website
- Email header

05-Favicons/    (square, for browser tabs and apps)
──────────────────────────────────────────────────────────────────
Square versions optimized for browser tabs, app icons, and PWA

Sizes: 16, 32, 48, 64, 128, 192, 256, 512 px
Variants: 4 color combinations

USE FOR:
- xanvor.com favicon (use 16, 32, 48 px)
- iOS app icon (use 192, 512 px)
- Android Chrome icon (use 192, 512 px)
- Windows tile icon (use 256 px)

WEBSITE FAVICON HOW-TO:
Upload the 4 sizes (16, 32, 48, 192) named as:
  favicon-16x16.png
  favicon-32x32.png
  favicon-48x48.png
  apple-touch-icon.png  (192px)

Add to your <head>:
  <link rel="icon" type="image/png" sizes="32x32" href="/favicon-32x32.png">
  <link rel="icon" type="image/png" sizes="16x16" href="/favicon-16x16.png">
  <link rel="apple-touch-icon" sizes="192x192" href="/apple-touch-icon.png">

──────────────────────────────────────────────────────────────────
07-SVG-Source/    (vector files for designers and printers)
──────────────────────────────────────────────────────────────────
Master SVG vector files for any variant. Give these to professional
printers, sign-makers, embroiderers, or designers. Vector means it

USE FOR:
- Send to printer for brass foil die-making

- Open in Figma / Illustrator for further design work
- Convert to other formats (EPS, PDF) as needed

═══════════════════════════════════════════════════════════════════
DO'S AND DON'TS
═══════════════════════════════════════════════════════════════════

✓ DO use these files as provided — they are master assets
✓ DO scale up from larger sizes (use 2048px for any large output)
✓ DO use the brass bead — it's the brand's signature
✓ DO maintain clear space around the mark (= height of N's stem)

✗ DON'T italicize, slant, or rotate the ligature
✗ DON'T add drop shadows, outlines, or 3D effects
✗ DON'T recolor individual parts of the mark
✗ DON'T use the mark below 16px (favicon minimum)
✗ DON'T stretch the mark — scale proportionally only
