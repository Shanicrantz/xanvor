# XANVOR

> Fine objects, bound by hand.

Brand site for XANVOR — a premium handcrafted homeware atelier based in Moradabad, India. Sculptural metal objects in copper, brass, stainless steel, iron, and aluminium.

---

## Stack

- Single-file static site — `index.html`
- No build step, no dependencies
- Google Fonts: Fraunces, Cormorant Garamond, Space Grotesk, JetBrains Mono
- Inline SVG wordmark and footer marks
- Hosted on Netlify, deployed continuously from this repo

## Local preview

```sh
# Any static server works. Examples:
python3 -m http.server 8000
# or
npx serve .
```

Then open http://localhost:8000

## Deployment

Pushes to `main` auto-deploy via Netlify. Production URL: https://xanvor.netlify.app (custom domain `xanvor.com` to follow).
